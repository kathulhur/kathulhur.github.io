Design system · v0.1

The rules that keep this portfolio calm, readable, and unmistakably mine.

A living reference for the visual language. Change a value in tokens.css and it ripples everywhere — this page and the portfolio both consume the same tokens.

Principles

Five commitments the whole system is built to honor.

Space groups things

Related elements are bound by proximity and rhythm — not by drawing a box around them. Whitespace is the primary layout tool.

Borders are intentional

No border is used to "just emphasize." When a cue is needed, prefer a short line on one side over a full enclosure.

One identity color

Indigo carries the brand. It appears on links, focus, one hero keyword, and the featured work — never as background noise.

One typeface

Inter throughout. Hierarchy comes from size, weight, and space — not from mixing families.

Personality, not decoration

The projects highlight is the moment of flair. Everywhere else stays composed so it never competes.

Color

A neutral ink-on-paper base with a single indigo identity. Accents are functional and rare. Toggle the theme (top right) to see both modes.

Borders — the intentional-only rule

Borders never do the job that space should. When emphasis is genuinely needed, a side line beats a box.

Do

Use a short line on one side to mark a relationship — timeline threads, section eyebrows, quote emphasis.

A side line points without enclosing.
Don't

Wrap an element in a full border just to make it "stand out." That's a lazy box — use surface tint or space instead.

A full box shouting for attention.
Do

Use one honest hairline as a structural divider (e.g. footer top, doc section breaks) where a real boundary exists.
